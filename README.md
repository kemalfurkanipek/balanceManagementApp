# Balance Management App

Balance Management is a web application that allows users to manage their balances, create coupons, and make payment transactions.

## Features

- User registration and login
- Viewing and updating balances
- Creating coupons
- Increasing balance through payments
- Viewing created coupons
- Changing the application's language
- Logging out of the application

## User Guide

1. If you don't have an account, register first.
2. After registering, log in to access the application.
3. **Main Features:**
   - **View Balances:** See your current balance on the homepage.
   - **Create Coupons:** Create coupons if you have sufficient balance.
   - **Increase Balance:** Use payment methods to add balance if it's insufficient.
   - **Language Settings:** Change the application's language in the settings section.
   - **Log Out:** Securely log out of your account.

## Technologies Used

- **React:** For building the user interface
- **Redux:** For state management
- **Tailwind CSS:** For design and styling
- **Firestore:** For database and user data management
